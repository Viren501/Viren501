### 📂 Highlighted Projects

#### 1. Job Offer Acceptance Prediction
An end-to-end machine learning pipeline designed to predict candidate offer acceptance. 
* **Live App:** [https://jobofferacceptanceprediction.streamlit.app/](https://jobofferacceptanceprediction.streamlit.app/)
* **Details:** Engineered features from behavioral indicators and explicitly mapped target variables (`Joined` = 1, `Not Joined` = 0). Benchmarked multiple classification algorithms and identified **Random Forest** as the absolute best-performing model for this specific predictive accuracy.

#### 2. Adaptive & Collaborative Traffic Control System
An IoT-based monitoring system for real-time traffic density forecasting and incident detection.
* **Details:** Built using real-time sensor data and the Random Forest algorithm to forecast traffic and prioritize emergency vehicles via V2X communication. Developed as part of the Smart India Hackathon (SIH) 2025.

#### 3. Car Average Prediction
A predictive model to estimate car mileage based on key vehicle specifications.
* **Details:** Performed comprehensive exploratory data analysis using Pandas and NumPy. Visualized key feature correlations with Matplotlib and Seaborn, and applied Linear Regression using Scikit-Learn to predict outcomes.
